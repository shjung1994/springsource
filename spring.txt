# sql 구문 실행
# DDL: create, alter, drop
spring.jpa.hibernate.ddl-auto= 5가지 종류
1) create: 기존 테이블 삭제 후 테이블 생성
2) create-drop: 기존 테이블 삭제 후 테이블 생성, 종료 시점에 테이블 삭제
3) update: 변경된 부분만 적용
4) validate: 엔티티와 테이블 정상 매핑 확인(스테이징, 운영환경)
5) none: 사용하지 않음(스테이징, 운영환경)
